# Weather App

> A weather app that pulls from the OpenWeatherMap API to allow users to search for and view the forecast in cities worldwide.

## Built With

- HTML
- CSS
- JavaScript

## Live Demo

[Weather App](https://karayamanemre.github.io/weather-app/)

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

   - Please install git in your computer.
   - Also install a code editor.
   - A web browser to view the output.

### Setup

   - To clone my repository run this command `git clone https://github.com/karayamanemre/weather-app.git`
